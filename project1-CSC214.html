Guessing Game

This guessing game is built in Kotlin in Android Studio. It contains several tricky questions and gives people two choices under each question. The ultimate goal is to answer all the questions correctly and enjoy the game. The game is designed in a quiz form, but it needs users to explore the app more to achieve 100% correctness.
