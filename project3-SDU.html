Train Ticket Management System

This is simple Train Ticket Management System developed in Chinese when I attended Study Local Program in China in Fall 2020. The project involves SQL for data storage and data accessing, JAVA for basic implementation, and HTML for web page setup. It achieves the goal for inserting, modifying, and deleting data for both administrators and users. Pages in white background are for administrators, while pages in green are for users.
