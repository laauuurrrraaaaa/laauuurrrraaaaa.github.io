Train Ticket Management System

This Train Ticket Management System was developed in Chinese in Fall 2020 when I attended the Study Local Program at Shandong University in China. The project involves the use of SQL for database, Java for back-end implementation, and HTML and CSS for web page display.

The system has an administrator view and a user view. The two views are visually different from their background colors. Administrator view has a white background, while the user view has a slight green background. Both start with login pages. The administrators are allowed to view, modify, insert, and delete train and ticket information under administrator view. And the users are allowed to view train and ticket information, buy tickets with their identification, check order details, cancel orders, and search by train ID under user view. The website provides an easy way for people to manipulate the database visually.
